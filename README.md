# Преобразование XML с использованием XSLT 1.0

Данный проект содержит XML-документ и XSLT-преобразование, выполняющее выборку данных с использованием ключей (`xsl:key`) и XPath 1.0.

## Состав проекта

- **courses.xml** — основной XML-документ со списком курсов, ключевых слов и преподавателей.
- **1info.xsl** — XSLT-преобразование (XSLT 1.0), применяемое к XML-документу.

## Функциональные возможности XSLT-преобразования

Преобразование формирует HTML-страницу, содержащую:

1. **Список курсов, которые читает преподаватель Борисов И.О.**
2. **Список курсов, в которых рассматривается тема XML**
3. **Список курсов, которые одновременно:**
   - читаются преподавателем Борисовым И.О.,
   - включают тему XSLT.

При реализации используются:
- механизм ключей `xsl:key`,
- выборка по значениям с помощью XPath 1.0,
- фильтрация результатов с помощью предикатов.
